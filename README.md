### Prerequisites

Buf:
```bash
sudo curl -sSL "https://github.com/bufbuild/buf/releases/download/v1.57.0/buf-$(uname -s)-$(uname -m)" -o "/usr/local/bin/buf"
sudo chmod +x "/usr/local/bin/buf"
```
